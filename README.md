# 价表管家

一个浏览器本地运行的价格表整理工具。将 `index.html` 用浏览器打开即可使用；也可以在此目录启动任意静态文件服务器。

## 已实现

- 导入 `.xlsx` / `.xls` / `.csv` / `.tsv` 价格表
- 自动识别常见中文字段：商品名称、货号、分类、规格、单位、价格、供应商、生效日期
- 按货号去重；没有货号时按“商品名称 + 规格 + 供应商”合并
- 新文件持续追加到浏览器中的商品库；相同商品会更新为新价格
- 自动按关键词辅助分类，可手动修改商品分类
- 搜索、分类筛选、价格排序、商品编辑
- 分类管理：新建、重命名、删除
- 导出“全部商品”以及按分类拆分的多个 Excel 工作表
- 导入记录与新增/更新/跳过统计

## 价格表建议字段

第一行请放表头，支持的常见列名包括：商品名称、货号、分类、规格、单位、价格、供应商、生效日期。列名不完全一致时，商品名称仍建议保留。

数据默认保存在当前浏览器的本地存储中，不会自动上传到服务器。清除浏览器站点数据或点击“清空数据”会删除本机数据；如需长期保存，定期导出 Excel。

## DeepSeek 智能分类版

仓库中的 `api/classify.js` 是 Vercel 后端接口。部署到 Vercel 后，在项目的 Environment Variables 中添加 `DEEPSEEK_API_KEY`，并重新部署。导入价格表后，网页会只发送商品名称和规格到 `/api/classify`，由 DeepSeek 判断分类；价格、数量和供应商不会发送。
